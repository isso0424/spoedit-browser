# spoedit-browser
The playlist editor for spotify.

## Usage
1. Launch auth/main.go.
2. Input ClientID/SecretID(set callback url: `http://localhost:8080/callback`).
3. Login to spotify.
4. Execute `yarn` and `yarn start`.
5. You can use this application!!!
